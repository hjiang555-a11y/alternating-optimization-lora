## 三、回答了什么问题

### 已回答 ✅

| 问题 | 答案 |
|------|------|
| 2×2 析因能否解耦优化器×参数形态？ | **能**。交互效应 >1000 PPL 证明必须解耦。 |
| LoRA 在低步数下是否主导？ | **是**。5-30× PPL 改善, 5/5 架构。 |
| AltOpt 是否收敛？ | **是**, 但非单调。OPT gap 82k→7k (7.8× 收缩, 800步)。 |
| ALS 为什么慢？ | 消化期 ~50-150 SGD 步/ALS cycle。Reconstruction loss ~10⁵ vs CE ~2-3。 |
| 扰动有帮助吗？ | 短期改善 eval PPL 但恶化 train loss (RWP trade-off)。 |

### 部分回答 ⚠️

| 问题 | 状态 |
|------|------|
| ALS 何时值得？ | 消化期瓶颈量化。**新视角**: AdamW 过拟合 → ASP 隐式正则化有价值 |
| AdamW plateau 是收敛还是过拟合？ | **已确认: 过拟合**。400-1600 samples 均出现, 200→400步 eval loss 持续上升 |
| **公平 gap** (AdamW 最佳 vs ASP) | **~4.4 loss (78 PPL)**, 远小于原始 1200步 gap (3527 PPL) |
| 最优 ALS:SGD 比？ | 1:20 在 50 步消融中最好。>200 步未确定。 |

### 未回答 ❌

| 问题 | 为什么 |
|------|--------|
| AltOpt 最终能否超越 AdamW？ | 交叉点 >>2000 步。受限于 CPU 计算时间。 |
| ALS+LoRA 协同在长步数是否存在？ | 仅测试到 400 步。需 >500 步。 |
| 7B+ 模型上结论是否成立？ | DeepSpeed ready, 模型未下载。 |
